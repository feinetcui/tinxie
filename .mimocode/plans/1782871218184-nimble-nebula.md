# Agnes AI vs SenseNova 对比

## 对比表

| 特性 | Agnes AI (agnes-2.0-flash) | SenseNova 6.7 Flash-Lite |
|------|---------------------------|--------------------------|
| **多模态支持** | ✅ 支持图像输入 | ✅ 原生多模态架构 |
| **图像理解** | ✅ OCR + 手写识别 | ✅ OCR + 图表解读 + 图像理解 |
| **国内访问** | ⚠️ 需确认 | ✅ 国内平台，稳定访问 |
| **上下文长度** | 未明确 | 256K tokens |
| **免费额度** | 有免费额度 | 需确认 |
| **API 兼容性** | OpenAI 兼容格式 | OpenAI 兼容格式 |
| **模型特点** | 轻量级 | 轻量高效，办公场景增强 |
| **Token 效率** | 一般 | 更优，成本更可控 |

## 推荐：SenseNova 6.7 Flash-Lite

**理由：**
1. **国内平台** - 访问稳定，无需担心网络问题
2. **原生多模态** - 专为图像理解设计，OCR 效果更好
3. **Token 效率更高** - 复杂任务成本更可控
4. **上下文更长** - 256K tokens，处理长文本更从容
5. **办公场景增强** - 稳定支撑复杂长链路任务

## 需要修改的文件

### 1. `src/worker.js`
修改 `callAgnesAI` 函数：
- API 端点：`https://api.sensenova.cn/v1/chat/completions`
- 模型名称：`sensenova-6.7-flash-lite`
- API Key 配置方式不变

### 2. Cloudflare Secret
- 设置新的 Secret：`SENSENOVA_API_KEY`
- 保留 `AGNES_API_KEY` 作为备用

## 验证步骤
1. 部署更新后的代码
2. 测试 OCR 识别功能
3. 测试手写识别功能
4. 确认无 500 错误
